## Environment
- Ubuntu 18.04
- Python 3.6
- Cuda 10.1

## Installation  

1. Clone this repository

```
git clone https://github.com/kyuyeonpooh/STTN.git
cd STTN/
```

2. Pull remote branch
```
git pull origin demo
```

3. Install requirements

```
pip install -r requirements.txt
```

4. Run the code
```
python train.py
```
